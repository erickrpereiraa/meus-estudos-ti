# Técnicas de Desenvolvimento de Algoritmos

Exercícios e atividades voltadas ao desenvolvimento do raciocínio lógico e construção de algoritmos.

##  Conteúdos praticados
- Variáveis e tipos de dados
- Estruturas condicionais (if/else)
- Estruturas de repetição (for, while)
- Vetores e matrizes
- Resolução de problemas com lógica

##  Exercícios disponíveis

### **Exercício-Java-e-Fluxograma**

#### **Java**

- **Exercício – Aprovação de Emprestimo:** 
  Aplicação em Java que recebe dados financeiros do usuário e verifica se ele atende aos requisitos para aprovação de um empréstimo. Prática de operadores lógicos, cálculos e estruturas de decisão.

- **Exercício – Consumo de Combustível:** 
  Programa em Java que calcula o consumo médio de combustível de um veículo com base na distância percorrida e na quantidade de combustível gasto. O exercício pratica operações matemáticas e exibição de resultados no console.

- **Exercício – Desconto:** 
  Programa em Java que calcula o valor final de um produto após a aplicação de um desconto percentual. O exercício pratica o uso de operadores matemáticos e variáveis para realizar cálculos financeiros simples.

- **Exercício – Fast-Food:** 
Programa em Java que simula um sistema simples de pedidos de fast food, onde o usuário escolhe itens de um cardápio.

#### **Fluxogramas**

- **Fluxograma – Aprovação de Emprestimo:** 
Fluxograma que representa o processo de decisão para aprovação de um empréstimo com base em dados financeiros informados pelo usuário. O diagrama demonstra o uso de decisões condicionais para verificar se os critérios exigidos são atendidos.

- **Fluxograma – Consumo de Combustível:**
  Fluxograma que demonstra o cálculo do consumo médio de combustível a partir da distância percorrida e da quantidade de combustível utilizada. O diagrama representa operações matemáticas e exibição do resultado.
  
- **Fluxograma – Desconto:**
  Fluxograma que descreve o processo de cálculo do valor final de um produto após a aplicação de um desconto percentual. O diagrama representa entrada de dados, cálculo e saída do valor com desconto.
 
- **Fluxograma – Fast-Food:**
  Fluxograma que representa o funcionamento de um sistema simples de pedidos em um fast food.

##  Objetivo
Desenvolver a base lógica necessária para a programação e resolução de problemas computacionais.
