# Programação de Computadores

Repositório com exercícios práticos desenvolvidos durante meus estudos de Programação de Computadores na graduação em Ciência da Computação.

## Conteúdos praticados
- Lógica de programação  
- Uso de variáveis  
- Estruturas condicionais (if/else)  
- Operações matemáticas  
- Entrada e saída de dados  

## Exercícios disponíveis

- **Exercício – Calcular inverso:** 
  Programa que recebe um número inteiro de três algarismos, calcula seu inverso (número ao contrário) e exibe a soma entre o valor original e o invertido.
  
- **Exercício – Contar caracteres:** 
  Programa que recebe um texto do usuário e conta quantas vezes cada caractere aparece, ignorando diferenças entre letras maiúsculas e minúsculas.

- **Exercício – Extrair dominio:** 
  Programa que recebe um endereço de e-mail e identifica o domínio ao qual ele pertence.
  

## Objetivo
Praticar os fundamentos da programação e desenvolver o raciocínio lógico para a resolução de problemas computacionais.
