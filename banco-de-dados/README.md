# Banco de Dados

Repositório com exercícios práticos desenvolvidos durante meus estudos de Banco de Dados na graduação em Ciência da Computação.

##  Conteúdos praticados
- Modelagem de dados
- Criação de tabelas
- Inserção de registros
- Consultas SQL (SELECT, filtros, contagem)

##  Exercícios disponíveis

- **Exercício – Sistema de Livraria:** 
  Criação de tabela e consultas simulando o cadastro de livros.

- **Exercício – Cadastro de Produtos:** 
  Criação de tabela de produtos e consultas com filtros.

##  Objetivo
Praticar os fundamentos de banco de dados relacional e comandos SQL, aplicando conceitos vistos em aula.
